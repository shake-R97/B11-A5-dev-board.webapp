Question-3: What is event delegation in the DOM?

Event Delegation in the DOM is a technique where we add single eventListener to it's parent to handel any of it's child element's instead of giving event handler to individual child

Question-4: What is event bubbling in the DOM?

Event Bubbling is when an event happen in a child element it also triggers it's parents and ancerstors element like a bubble keeps bubbling up, it helps us control other's element at the same times.
